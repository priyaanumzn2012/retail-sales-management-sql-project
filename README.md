# Retail Sales Management System – SQL Project

## Project Overview

This project is a SQL-based Retail Sales Management System designed to simulate real-world retail business operations and analytical reporting.

The project includes:

* Database creation
* Table relationships
* Data insertion
* CRUD operations
* Analytical queries
* Joins
* Subqueries
* Window Functions
* Views
* Variables and CASE statements

The system helps analyze:

* Customer behavior
* Product performance
* Employee performance
* Sales operations
* Order trends
* Business KPIs

---

# Database Schema

The database contains the following tables:

1. Customers
2. Products
3. Employees
4. Orders
5. OrderDetails

These tables are connected using relational database concepts.
